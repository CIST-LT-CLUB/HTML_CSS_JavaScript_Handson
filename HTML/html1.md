# HTML Handson
## 1. HTMLの雛形
HTMLの形はこれだ！

```
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <title>タイトル</title>
  <meta name="description" content="サイトの説明を記載します">
  <link rel="icon" href="favicon.ico">
</head>
<body>
  <!-- ここにコンテンツを記載します -->
</body>
</html>
```

これをまずは覚えよう！！  
困ったらこれをコピペしよね。
